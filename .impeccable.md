## Design Context

### Users
Cybersecurity students and staff at KKU (Khon Kaen University, Thailand). Use cases: secure file sharing between colleagues, hash verification, URL threat scanning, password analysis, AI-assisted image location detection. Used in an academic/professional security context.

### Brand Personality
Precise · Trustworthy · Approachable

### Aesthetic Direction
"Precision Security" — clean, editorial interface. Warm stone neutrals with a single teal accent (#0d9488). Inspired by: 1Password, Cloudflare Dashboard, Linear. Anti-reference: generic dark-mode neon, AI cyan/purple gradients, glassmorphism. Light mode only.

### Design Principles
1. Single accent color — teal only, no gradient "AI palette"
2. Type-led hierarchy — Sora for headings, DM Sans for body, JetBrains Mono only for actual technical data
3. Cards only where content needs containment — not as decoration everywhere
4. Generous whitespace — space creates clarity and trust
5. Clean & approachable — professional without being intimidating
